# Hanna & Chuck Kanban Board

A simple Kanban board for task management between Hanna (AI assistant) and Chuck.

## Features
- Drag & drop task management
- Assign tasks to Hanna, Chuck, or Both
- Priority levels (High/Medium/Low)
- Labels for categorization
- Four columns: Backlog, In Progress, Review, Done
- Auto-saves to browser localStorage

## Usage

Open `index.html` in any modern web browser. No server required!

## How to Add Tasks

1. Click "+ Add Task"
2. Enter title and description
3. Assign to Hanna, Chuck, or Both
4. Set priority and labels
5. Click "Save Task"

## Task Management

- Drag tasks between columns to change status
- Click "Edit" to modify a task
- Click "Delete" to remove a task

## Data Storage

All tasks are stored in your browser's localStorage. Data persists across sessions.
